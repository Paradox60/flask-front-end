Работа с linear-gradient в CSS

Исчерпывающее руководство по созданию линейных градиентов

Что такое linear-gradient?

Linear-gradient - это CSS-функция, которая создает изображение, состоящее из постепенного перехода между двумя или более цветами вдоль прямой линии.

background: linear-gradient(direction, color-stop1, color-stop2, ...);

Базовый синтаксис

1. Направление градиента

Направление можно задавать ключевыми словами или градусами:

to right

linear-gradient(to right, #ff9a9e, #fad0c4)

45deg

linear-gradient(45deg, #a1c4fd, #c2e9fb)

to bottom right

linear-gradient(to bottom right, #ffecd2, #fcb69f)

2. Цветовые остановки (Color stops)

Вы можете контролировать положение цветовых переходов:

С указанием позиций

linear-gradient(to right, #4facfe 0%, #00f2fe 50%, #a6ffcb 100%)

С ограниченным переходом

linear-gradient(to right, #ff9a9e 20%, #fad0c4 80%)

Практические примеры

Радужный градиент

linear-gradient(90deg, rgba(131,58,180,1) 0%, rgba(253,29,29,1) 50%, rgba(252,176,69,1) 100%)

Повторяющийся градиент

repeating-linear-gradient(45deg, #3f87a6, #ebf8e1 15%, #f69d3c 20%)
');">

Наложение на изображение

linear-gradient(to bottom, transparent 0%, rgba(0,0,0,0.5) 100%), url(image.jpg)

Советы и лучшие практики

  • Всегда указывайте fallback цвет для браузеров, которые не поддерживают градиенты
  • Используйте инструменты типа CSS Gradient Generator для быстрого создания сложных градиентов
  • Помните о доступности - убедитесь, что текст остается читаемым на фоне градиента
  • Экспериментируйте с прозрачностью (rgba) для создания сложных наложений

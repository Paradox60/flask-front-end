Переключатель языков

Как использовать в Django?

1. Убедитесь, что в вашем проекте Django настроена поддержка интернационализации (i18n).

2. Добавьте этот код в ваш шаблон, убедившись, что тег {% csrf_token %} включен в форму.

3. Замените # в атрибуте action на {% url 'set_language' %}.

4. Добавьте логику для динамического отображения текущего языка с помощью {% get_current_language %}.

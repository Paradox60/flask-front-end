1. Подключение JavaScript к странице

JavaScript можно подключить к HTML-странице двумя способами:

Внешний файл

<script src="script.js"></script>

Встроенный скрипт

<script>
  console.log("Hello, World!");
</script>

2. Переменные и типы данных

В JavaScript есть три способа объявления переменных:

var oldVariable = "Устаревший способ";
let changeableVariable = "Можно изменить";
const constantVariable = "Нельзя изменить";

Основные типы данных:

let string = "Текст"; // Строка
let number = 42; // Число
let boolean = true; // Логический тип
let array = [1, 2, 3]; // Массив
let object = {name: "John", age: 30}; // Объект

3. Операторы и выражения

JavaScript поддерживает различные операторы:

Арифметические операторы

let sum = 5 + 3; // 8
let difference = 10 - 4; // 6
let product = 3 * 4; // 12
let quotient = 20 / 5; // 4
let remainder = 10 % 3; // 1

Операторы сравнения

let equal = 5 == "5"; // true (нестрогое сравнение)
let strictEqual = 5 === "5"; // false (строгое сравнение)
let notEqual = 5 != 10; // true
let greaterThan = 10 > 5; // true

4. Условные конструкции if/else

Конструкции if/else позволяют выполнять код в зависимости от условий:

let temperature = 25;

if (temperature > 30) {
  console.log("Жарко");
} else if (temperature > 20) {
  console.log("Тепло");
} else {
  console.log("Прохладно");
}

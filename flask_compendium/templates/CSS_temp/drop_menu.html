Выберите вариант текста

Выберите вариант из выпадающего списка, чтобы увидеть другой текст.

Первый вариант текста

Это содержимое показывается при выборе первого варианта в меню. Здесь может быть любая информация, соответствующая выбранному пункту.

Второй вариант текста

При выборе второго варианта отображается этот текст. Каждый вариант может содержать совершенно разную информацию и даже различную структуру.

  • Элемент списка 1
  • Элемент списка 2
  • Элемент списка 3

Третий вариант текста

Этот текст появляется при выборе третьего варианта. Механизм позволяет легко переключаться между различными блоками контента на странице.

Такой подход полезен для создания интерактивных интерфейсов без перезагрузки страницы.

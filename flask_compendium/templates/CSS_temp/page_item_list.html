Font-Style и креативные текстовые эффекты

Изучите возможности оформления текста с помощью CSS

Font-Style

Свойство font-style определяет начертание шрифта — обычное, курсивное или наклонное.

Это текст с normal начертанием (обычный).

font-style: normal;

Это текст с italic начертанием (курсив).

font-style: italic;

Это текст с oblique начертанием (наклонный).

font-style: oblique;

Font-Weight

Свойство font-weight определяет насыщенность шрифта.

Тонкое начертание (100)

font-weight: 100;

Обычное начертание (400 - normal)

font-weight: 400;

Жирное начертание (700 - bold)

font-weight: 700;

Очень жирное начертание (900)

font-weight: 900;

Text-Transform

Свойство text-transform управляет регистром текста.

Этот текст преобразован в верхний регистр

text-transform: uppercase;

Этот текст преобразован в нижний регистр

text-transform: lowercase;

этот текст преобразован в регистр заголовка

text-transform: capitalize;

Letter-Spacing

Свойство letter-spacing определяет межбуквенное расстояние.

Широкое межбуквенное расстояние

letter-spacing: 5px;

Обычное межбуквенное расстояние

letter-spacing: normal;

Узкое межбуквенное расстояние

letter-spacing: -1px;

Text-Shadow

Свойство text-shadow добавляет тень к тексту.

Простая тень для текста

text-shadow: 2px 2px 4px rgba(0,0,0,0.3);

Множественные тени

text-shadow: 1px 1px 2px black, 0 0 25px blue, 0 0 5px darkblue;

Неоновый эффект

text-shadow: 0 0 5px #fff, 0 0 10px #fff, 0 0 15px #0073e6, 0 0 20px #0073e6;

Креативные эффекты

Современный CSS позволяет создавать впечатляющие текстовые эффекты.

Градиентный текст

background: linear-gradient(45deg, #f3ec78, #af4261);
-webkit-background-clip: text;
background-clip: text;
color: transparent;

Текст с обводкой

color: white;
-webkit-text-stroke: 2px #3498db;

Наведите на меня (переменный шрифт)

font-variation-settings: 'wght' 400;
transition: font-variation-settings 0.5s ease;
/* При наведении: */
font-variation-settings: 'wght' 800;

Что такое SVG?

SVG (Scalable Vector Graphics) — это язык разметки на основе XML для описания двухмерной векторной графики. В отличие от растровых изображений (JPEG, PNG), SVG-изображения не теряют качества при изменении размера и могут быть стилизованы с помощью CSS и управляемы с помощью JavaScript.

Масштабируемость

Изображения SVG можно увеличивать и уменьшать без потери качества, что идеально для адаптивного дизайна.

Маленький размер файла

Для простых изображений SVG обычно имеют меньший размер файла по сравнению с растровыми аналогами.

Доступность для стилей и скриптов

Элементы SVG можно стилизовать с помощью CSS и анимировать с помощью JavaScript.

Доступность

SVG-изображения могут содержать текстовые описания для улучшения доступности.

Базовый синтаксис SVG

Тег <svg> создает контейнер для SVG-графики. Основные атрибуты:

  • width и height — определяют размеры области SVG
  • viewBox — определяет систему координат и соотношение сторон
  • xmlns — пространство имен XML (обязательный атрибут)

Пример кода:

<svg width="200" height="100" viewBox="0 0 200 100" xmlns="http://www.w3.org/2000/svg">
  <rect x="10" y="10" width="180" height="80" 
        fill="lightblue" stroke="blue" stroke-width="3"/>
  <text x="100" y="55" text-anchor="middle" 
        font-size="20" fill="darkblue">
    Привет, SVG!
  </text>
</svg>

Основные фигуры в SVG

SVG предоставляет несколько базовых элементов для создания графики:

Элемент Описание Основные атрибуты
<rect> Прямоугольник x, y, width, height, rx, ry (закругления)
<circle> Круг cx, cy (центр), r (радиус)
<ellipse> Эллипс cx, cy, rx, ry (радиусы)
<line> Линия x1, y1, x2, y2 (координаты начала и конца)
<polyline> Ломаная линия points (список координат точек)
<polygon> Многоугольник points (список координат вершин)
<path> Произвольный путь d (команды пути)

Стилизация SVG

Элементы SVG могут быть стилизованы с помощью атрибутов или CSS. Основные свойства стилей:

  • fill — цвет заливки
  • stroke — цвет обводки
  • stroke-width — толщина обводки
  • opacity — прозрачность
  • transform — преобразования (поворот, масштаб и т.д.)

Пример с CSS-стилизацией:

<style>
  .styled-rect {
    fill: #ff6b6b;
    stroke: #c44;
    stroke-width: 5;
    opacity: 0.8;
    transform: rotate(10deg);
  }
  .styled-circle {
    fill: #4ecdc4;
    stroke: #2a8c85;
    stroke-width: 3;
  }
</style>

<svg width="200" height="150">
  <rect class="styled-rect" 
        x="20" y="20" width="100" height="80" rx="10"/>
  <circle class="styled-circle" 
          cx="150" cy="70" r="40"/>
</svg>

Практические примеры использования

1. Иконки и логотипы

SVG идеально подходит для иконок, которые нужно масштабировать под разные размеры экрана.

2. Простая диаграмма

Рекомендации по использованию

  • Используйте SVG для иконок, логотипов, диаграмм и простой графики
  • Для сложных фотографических изображений лучше подходят растровые форматы (JPEG, PNG)
  • Оптимизируйте SVG-файлы с помощью инструментов, таких как SVGO
  • Используйте атрибут viewBox для создания отзывчивых SVG-изображений
  • Для улучшения доступности добавляйте элементы <title> и <desc>
  • Используйте инлайновые SVG для простых изображений, которые нужно стилизовать или анимировать
